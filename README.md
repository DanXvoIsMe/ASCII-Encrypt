# ASCII Encrypt
Simple Encryption library for lua with support for encryption key.
This library has been designed for OpenComputers minecraft mod!

# Installation
1. For opencomputers (openos):
```bash
wget https://raw.githubusercontent.com/DanXvoIsMe/ASCII-Encrypt/main/asciiencryption.lua /lib/asciiencryption.lua
```
2. For others:
Just download a lua file and use it as library.
